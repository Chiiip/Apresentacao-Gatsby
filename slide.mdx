SSG + framework React

<Notes>
O Gatsby é um SSG, sigla em inglês para gerador de sites estáticos, e também um framework React que permite a criação de páginas para
a web, com o diferencial de o framework te entregar já prontas algumas otimizações que vão deixar o seu website mais rápido.
Antes de detalhar as features e as coisas presentes no próprio Gatsby que garantem essas vantagens, eu gostaria de primeiro explicar
como funciona a geração de páginas estáticas do Gatsby e o que essa geração de páginas tem de diferente de uma página feita com
Create React App e com Next. Perguntar quem conhece ou já trabalhou com Next
</Notes>

<h2>CRA ou Next</h2>
</Appear>

<Notes>
Aqui é bom destacar que apesar das funcionalidades do Gatsby não serem em sua maioria focadas para aplicações dinâmicas, nada impede que
você use Gatsby no seu site/app que seja muito dinâmico para poder aproveitar. E aí eu falo por experiência própria, pois a única aplicação
que eu já trabalhei em produção com Gatsby é um backoffice praticamente todo dinâmico, e a decisão de colocar Gatsby no projeto foi somente
para 1 - ter uma performance melhor no carregamento das telas, 2 - ter a otimização de imagens estáticas usando o Gatsby-image
</Notes>
